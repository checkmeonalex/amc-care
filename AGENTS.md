# Icons
Always use `lucide-react` for UI icons — it is already installed. Never write custom inline SVG paths for icons that exist in Lucide. For brand/social icons (Facebook, Instagram, etc.) that Lucide does not carry, use inline SVG only. Logo SVGs and data-visualisation SVGs (charts, illustrations) are exempt.
